# win-screencapture
Window Auto Screen Capture Tool

## Purpose
- Capture screenshots of a window at every frame.

## Code generation
- Gemini 2.0
- VSCode & Github Copiot

## Remarks
 - Use D3D for the Video Option

## Issues
